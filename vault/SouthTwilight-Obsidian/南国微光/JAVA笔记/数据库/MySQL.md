**绪论：**

```
    MySQL是一种常用的数据库管理系统，主要学习MySQL系统中需要使用的SQL语言。

    SQL是一门标准通用的结构化查询语言。标准的SQL适合于所有的数据库产品。SQL属于高级语言，SQL语句具有现实的含义，可以直接解读。
```

**SQL语句分类:**

> 1.DQL(Data Query Languege)（数据查询语言）: 查询语句，凡是select语句都是DQL
> 2.DML(Data Manipulation Languege)（数据操作语言）： insert/delete/update 对表当中的数据进行增删改
> 3.DDL(Data Definition Languege)（数据定义语言）： create/drop/alter，对表结构的增删改
> 4.TCL(Transaction Contral Languege)（事务控制语言）： commit提交事务，rollback回滚事务。(T—Transaction：业务)
> 5.DCL(Data Contral Languege)（数据控制语言）: grant授权、revoke撤销权限等。
